# AWS Static Website Hosting Project

## Overview
This project demonstrates hosting a static website on AWS using Amazon S3.

## AWS Services Used
- Amazon S3
- IAM

## Architecture
User → Browser → Amazon S3

## Features
- Public static website
- HTML & CSS
- AWS Free Tier compatible
